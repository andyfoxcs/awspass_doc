**自定义ID**

字母和数字的组合。

> 自定义ID的使用场景：当使用[`流程模版管控`](<https://docs.awspaas.com/apps/com.actionsoft.apps.addons.processdeftpl/>)应用时，如果一个流程是从模板生成的，节点ID会发生变化；如果需要节点ID固定，可以在流程模板中自定义一个唯一标识，保证模板流程和派生流程节点唯一无变化
