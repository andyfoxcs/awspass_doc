# 图片标识

动态显示一个图片，这是一个私有封装。可支持数字、英文字母下划线和@公式。 运行时，保存数据后数据库中字段存储的是设计时图片标识名称和图片格式的组合

> 目前该UI组件移动端仅支持只读展示图片，不支持上传修改图片
