# 脚本

视图加载完成后的一种补偿事件。

**步骤**

  1. 打开视图配置界面
  2. 鼠标移动视图名称上，点击"高级选项"弹出侧边栏，点击"脚本"页签
  3. 直接写JavaScript代码，点击"保存"按扭，完成操作
