# 门户

AWS PaaS为开发者提供的门户操作接口。

> 以下给出该类API的常用部分，但不完整，请以相关文档为准

  * 获取用户可访问的菜单结构
  * 用户信息
    * 口令修改
    * 基本信息修改
    * 用户头像
    * 登录日志
  * 会话信息
    * 创建会话
    * 检查会话
    * 刷新会话
    * 注销会话
    * 在线列表

API类型 | 说明
