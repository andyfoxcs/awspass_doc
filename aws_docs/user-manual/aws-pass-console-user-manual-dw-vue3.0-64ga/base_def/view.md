# 视图

视图共分为以下几种类型

  * **流程应用视图** 针对工作流引擎流程
    * **普通视图** 显示指定数据源的所有数据列表
    * **我的待办视图** 只显示当前用户的待办任务数据列表(包含传阅、委托、沟通、流程通知等)，提供任务执行动作
    * **我的已办视图** 只显示当前用户的已办任务数据列表，提供任务收回、跟踪动作
    * **发起跟踪视图** 只显示当前用户发起流程实例的列表，提供催办、撤销动作

  * **数据应用视图** 没有流程控制，仅用于数据维护
    * **普通视图** 显示指定数据源的所有数据列表
