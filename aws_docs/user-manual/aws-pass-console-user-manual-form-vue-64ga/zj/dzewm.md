# 字段二维码

动态显示一个条形码/二维码图片，这是一个私有封装。该组件可以将指定值转换成条形码/二维码图片。

**标题**

参见单行[标题](<text.html#title>)

**展示方式**

条形码和二维码这两种方种展示方式

**显示类型**

只有条件码有显示类型。共四种显示类型`码和文字在上` `文字` `码` `码和文字在下`，默认选中`码和文字在上`

**编码信息**

一个字符串，通常该值由@公式提取表单动态值

参见单行[宽度](<text.html#wigth>)
