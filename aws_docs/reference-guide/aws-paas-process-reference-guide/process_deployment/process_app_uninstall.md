# 卸载流程应用

卸载应用是指应用从当前AWS PaaS环境卸载。为避免错误操作给企业带来损失，要完全卸载应用，请遵循如下步骤：

  * 执行卸载应用操作，应用将脱机转至`卸载`仓库
  * 从`卸载`仓库执行彻底卸载应用操作，应用和数据将物理删除

> 含有流程的应用彻底卸载后，与之相关的流程控制数据也将物理删除
