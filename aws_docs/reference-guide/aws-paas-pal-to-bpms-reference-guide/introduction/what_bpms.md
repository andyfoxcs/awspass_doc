# 什么是BPMS?

AWS BPMS(Business Process Management System)是AWS PaaS的重要服务组件。BPMS提供执行流程所需的服务，例如：

  * 流程建模
  * 流程引擎
  * 流程运维
  * SDK API
  * 流程客户端
  * 集成设施
  * 表单和存储
