# 组织

AWS PaaS为开发者提供的组织结构操作接口。

> 以下给出该类API的常用部分，但不完整，请以相关文档为准

  * 单位增、删、改、查
  * 部门增、删、改、查
  * 账户增、删、改、查
  * 角色增、删、改、查
  * 团队增、删、改、查
