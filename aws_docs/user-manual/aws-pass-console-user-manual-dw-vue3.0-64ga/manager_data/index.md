## 配置数据方案

配置数据方案是数据应用视图特有的，根据不同业务场景实现对数据的不同管理方式。

**步骤**

  1. 点击‘应用服务>业务模型’，展开导航树相关业务模型
  2. 点击"数据窗口"，右侧列出当前业务模型所有数据窗口列表（包含流程应 用视图和数据应用视图）
  3. 点击数据应用视图，打开视图配置界面
  4. 点击右上角"配置数据方案"链接，打开配置数据方案
