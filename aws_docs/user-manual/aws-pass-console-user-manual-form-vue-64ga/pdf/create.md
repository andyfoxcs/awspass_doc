# 新建

步骤：

1.进入AWS PaaS控制台，依次点击`应用服务>业务模型`  
2.左侧树选择业务模型库，点击`新建`按钮，打开`新建`窗口  
3.选择应用名称、业务模型分类和数据字典模型，确定，打开新建关联选择窗口  
4.输入字典名称确定  
5.在打开字典管理器窗口中配置相关信息，完成字典的创建
