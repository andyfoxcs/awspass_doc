# 邮箱 · AWS PaaS文档中心

# 邮箱

创建一个文本输入框，对标准INPUT type=TEXT元素封装，只能输入邮箱，以显示和修改被表单数据源绑定后的数据，自动通过平台各种权限配置控制其读、写、隐藏状态。
