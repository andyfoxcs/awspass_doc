# 从数据源新建表单

**步骤：**

  1. 点击`从数据源新建表单`弹出`选择数据源`框
  2. 选择主表、子表BO,点`确定`，打开表单设计器
  3. BO存储对应的字段在表单设计区已展示

---  
  
> 选择主子表BO列出的是当前应用与当前应用关联应用的相关BO
